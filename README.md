# Did
An node.js script for changing the wallpaper about your favorite topics.

## help 
Type:
`./run.js -h` *for help*
## install
In your Shell:
`npm install .`
`chmod +x ./run.js`
`./run.js -s` *to set the config*
`./run.js` *to run or add to the startup*
